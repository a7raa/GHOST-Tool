# 🤖 GHOST-Tool: My Personal AI Assistant
مساعد ذكي خاص يعمل على بيئة Termux باستخدام لغة Python.

## 🌟 عن المشروع (About)
هذا البرنامج هو نسخة مخصصة من الذكاء الاصطناعي (AI Assistant) مصممة لتعمل مباشرة من الهاتف عبر تطبيق Termux. تم برمجته ليكون مساعداً تقنياً في لغة بايثون ولينكس.

## 🚀 المميزات (Features)
- **ذكاء اصطناعي محلي:** يتفاعل معك بذكاء وسرعة.
- **دعم Termux:** متوافق تماماً مع بيئة أندرويد.
- **خفيف وسريع:** لا يحتاج لإمكانيات جهاز عالية.
